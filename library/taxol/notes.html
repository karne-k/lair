      Notes on Site

      I've tried to do my best in making this site compatible with most types and/or mutations of the standard web browsers but there may be some problems. In an attempt to help, I've included the following notes. If this is showing fine on your browser, then go back to the main page.

      • This site is designed for Netscape 3 or above (or Internet Explorer (I. E.), if you prefer it <wimper!>)
      • This page uses frames (yes, I know they're evil but I've tried to make them behave), so please turn them on. You can read the site without frames but you'll get about 3 separate browser windows.
      • All text is at +1 standard size, to make it easier to read - if this is too big (which might be a problem with I. E. which uses a larger default font) then set your default to 10 pts (or small, if you're using I. E.).
      • I've built this for a screen res. of 800 by 600 pixels. If you're using less then this, you will have problems with the reference frame (bottom right) - in that it will have a bottom 'scroll bar' that gets in the way. Sorry, there's not much I can do about that. Suggest that you make the browser window as big as you can.
      • Please have JavaScript activated.
      • Some pictures will need the Chime plugin. You can get this from mdl. This allows active 3D molecules.

      Right, that's it. Thanks and enjoy the rest of the site ];=8) If you have any problems / comments then yell at me on karne@innocent.com
